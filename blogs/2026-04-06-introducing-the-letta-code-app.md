Letta has released the Letta Code app, enabling users to build personalized agents that learn over time and work locally on your machine. The application features memory systems that evolve through interactions, allows switching between AI models while preserving agent context, and supports extensible skills to enhance agent capabilities. Downloads are available for macOS, Windows, and Linux.
