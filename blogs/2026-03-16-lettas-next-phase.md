Letta is transitioning to Letta Code, an open-source agent framework with persistent memory that works across all language models. The company is shifting from server-side features to client-side capabilities emphasizing computer use, memory management through git-backed files, and dynamic subagents, while sunsetting legacy tools like hardcoded memory operations and template systems by mid-April.
