Letta has launched Remote Environments, enabling users to interact with agents from chat.letta.com while they execute on locally registered machines. The feature allows a single agent to work across different platforms, including laptops, sandboxes, and remote VMs, while maintaining persistent memory and conversation history. Users can configure various permission modes to control how much autonomy agents have when executing remotely.
