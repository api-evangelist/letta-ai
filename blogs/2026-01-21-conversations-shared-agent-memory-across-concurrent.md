Letta has introduced a conversations API that enables agents to manage multiple concurrent conversations while maintaining a unified memory system. This allows a single agent to serve multiple users or channels simultaneously, with shared memories across all conversations. The feature simplifies agent deployment by eliminating the need to create separate agents for each user.
