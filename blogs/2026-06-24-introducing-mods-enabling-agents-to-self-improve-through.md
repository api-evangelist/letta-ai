Letta has introduced Mods, a feature enabling agents to modify their own runtime environment for self-improvement. These modifications can include adding tools and commands, adjusting how context is provided to models, and creating custom UI elements. The system allows agents to "learn a better harness" by adapting both their knowledge and the underlying execution environment through code generation and feedback-based optimization.
